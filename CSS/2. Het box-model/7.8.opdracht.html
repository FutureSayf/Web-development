Opdracht 7

Maak nu een blok met de volgende afmetingen:

  • width: 100px
  • height: 100px
  • margin: 5px
  • padding 5px
  • border-width: 5px

..Het Antwoord:

Let's Go!



dit is de content.


  1. Hoe breed is de box?
    • 100px + (5px + 5px * 2) = 120px
  2. Hoe hoog is de box?
    • 100px + (5px + 5px * 2) = 120px
  3. Wat merk je op met width en height?
    • Hier moeten nog de afmetingen van padding, margin bij opgetelt worden.
