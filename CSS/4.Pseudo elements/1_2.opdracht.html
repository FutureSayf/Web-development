Opdracht Nr. 1 & 2

Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen.
Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

Het Antwoord..

pseudo-elementen zijn als virtuele elementen die we kunnen behandelen als gewone HTML-elementen.
Het probleem is dat ze niet voorkomen in de documentboom of DOM.
Dit betekent dat we de pseudo-elementen niet echt typen, maar ze maken met CSS.

Hier een overzicht van voorkomende Pseudo elementen.

  • ::after

    Het, ::after pseudo-element geeft aan dat het laatste kind is van het geselecteerde element.
    Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de eigenschap inhoud.
    Het is standaard inline. Meer informatie zie de webpage: ::after

  • ::backdrop

    Het :: backdrop CSS pseudo-element is een box ter grootte van de viewport die direct wordt weergegeven onder elk element dat in volledig scherm wordt gepresenteerd.
    Dit omvat zowel elementen die in volledig scherm zijn geplaatst met behulp van de Fullscreen API en -dialog- elementen.

    Wanneer meerdere elementen in de modus Volledig scherm zijn geplaatst,
    wordt de achtergrond direct onder het voorste element van dit element getekend,
    en bovenop de oudere elementen op volledig scherm. Meer info zie de webpage over: ::backdrop

  • ::before

    In CSS maakt ::before een pseudo-element aan dat het eerste kind is van het geselecteerde element.
    Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de eigenschap inhoud.
    Het is standaard inline. Meer informatie zie de webpage over: ::before

  • ::cue

    Het :: cue CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element.
    Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media te stylen met VTT-tracks.
    Meer informatie hier over zie: ::cue

    De eigenschappen worden toegepast op de hele set cues alsof ze een enkele eenheid zijn.
    De enige uitzondering is dat de achtergrond en de eigenschappen voor de lange hand op elke keu afzonderlijk van toepassing zijn,
    om te voorkomen dat er boxen ontstaan ​​en onverwacht grote delen van de media worden verduisterd.

  • ::cue-region

    Het ::cue-region CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element.
    Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media te stylen met VTT-tracks.

    De eigenschappen worden toegepast op de hele set cues alsof ze een enkele eenheid zijn.
    De enige uitzondering is dat de achtergrond en de steno-eigenschappen ervan op elke cue afzonderlijk van toepassing zijn,
    om te voorkomen dat er boxen worden gemaakt en onverwacht grote delen van de media worden verduisterd.
    Meer informatie over dit onderwerp zie de webpage: ::cue-region.

  • ::first-letter

    Het ::first-letter CSS pseudo-element past stijlen toe op de eerste letter van de eerste regel van een blokniveau-element,
    maar alleen als deze niet wordt voorafgegaan door andere inhoud (zoals afbeeldingen of inline-tabellen).
    Meer informatie hier over zie de webpage: ::first-letter.
