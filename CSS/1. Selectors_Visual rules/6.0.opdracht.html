Selectors CSS

CSS-selector is het deel van een CSS-regel dat beschrijft met welke elementen in een document de regel overeenkomt.
Op de overeenkomende elementen wordt de opgegeven stijl van de regel toegepast.
We kunnen CSS selectors in 5 categories indelen.

  • Simple selectors
    • Type selector
    • Class selector
    • ID selector
    • Universal selector
    • Attribute selector
    • State selector

  • Combinator selectors
    • Adjacent selector
    • General selector
    • Child selector
    • descendant selector

  • Pseudo-class selectors
  • Pseudo-elemnets selectors
  • Attribute selectors

In dit overzicht zullen we er enkele toelichten.

Child selectors

seleceteren alleen items die direct verwand zijn van een lijst.
zie voorbeeld hieronder direct verwand van OL orderd list zijn alle LI's.
de Li's die onder UL komen zijn niet direct verwand.

De Child combinator (>) wordt tussen twee CSS-selectors geplaatst.
Het komt alleen overeen met die elementen die overeenkomen met de tweede selector
die de directe kinderen zijn van elementen die overeenkomen met de eerste.

Zie bron: developer.mozilla.org

Adjacent Selectors

De Adjacent combinator op hetzelfde niveau (+) scheidt twee selectors en komt alleen overeen met het tweede element
als het onmiddellijk volgt op het eerste element, en beide kinderen zijn van hetzelfde bovenliggende element.

Deze selectors kunnen u helpen styling op een contextuele manier toe te passen.
Misschien heb je een heleboel artikelen op een site. De meeste beginnen met "P"-elementen.
Maar sommige beginnen met een "ul", die een boven-en ondermarge hebben om er goed uit te zien in een artikel.
Maar als ze aan een artikel beginnen, ontstaat er een ongemakkelijke kloof.
De laatste selector hierboven zou die bovenmarge op nul zetten wanneer het een artikel begint (volgt meteen een titel).

Zie bron : developer.mozilla.org

General selectors

De algemene combinator tussen broers en zussen (~) scheidt twee selectors en komt overeen met alle iteraties van het tweede element,
die volgen op het eerste element (hoewel niet noodzakelijk onmiddellijk),
en zijn kinderen van hetzelfde bovenliggende element.

Zie bron: developer.mozilla.org
